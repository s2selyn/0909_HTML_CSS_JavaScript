        선택자 {
            font-family : 바꾸고싶은글꼴명;

font-size : 글꼴의 크기를 조절할 때 사용

        [ 표현법 ]

        선택자 {
            font-size : 크기에 해당하는 숫자(단위 : px / em / %);
        }

font-weight : 글꼴의 두께를 표현할 때 사용

        [ 표현법 ]
        선택자 {
            font-weight : normal / bold / bolder / lighter
                         / 100 ~ 900
        }
  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 굵게
  • 원래 굵기보다 더 얇게

font-style : 기울이고 싶을 때 사용

        [ 표현법 ]
        선택자 {
            font-style : normal / italic / oblique;
        }

        -> italic : 기울임 글꼴, oblique : 원래 폰트를 기울임
