innerHTML / innerText
자바스크립트에서 특정 요소(Element)가 가지고 있는 값(Content영역)을 가지고 온다거나(자바에서 getter 쓰는 방법)
또는 값(Content)를 변경하려고 할 때(자바에서 setter 쓰는 방법), innerHTML, innerText 라는 속성을 이용
=> 가지고 올때 : 요소객체.innerHTML / innerText
=> 변경할 때 : 요소객체.innerHTML / innerText = '변경할 값';

1) window.confirm('질문내용')

확인 버튼 클릭 시 true값, 취소 버튼 클릭 시 false값 반환(선택에 따라 다른 논리값 반환)

2) window.prompt('질문내용')

- 확인 버튼 클릭 시 텍스트 상자에 입력한 값을 반환
