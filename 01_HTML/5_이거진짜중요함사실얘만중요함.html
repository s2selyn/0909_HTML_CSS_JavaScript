form태그에 action속성은 꼭 써줘야지 꼭 써줘야겠다.
method=""
method속성은 필요하면 꼭 써줘야겠다. 근데 필요하면 써줘야겠다.
* action : 해당 form안에 사용자가 입력한 값들을 전달하면서 요청 보낼 URL을 작성
* method : 요청 전송 방식 지정하는 속성(GET방식(기본값)/POST방식)
> GET방식 : 요청 시 사용자가 입력한 값들이 URL에 노출
> POST방식 : 요청 시 사용자가 입력한 값들이 URL에 안보임
form태그 내부에 존재하는 submit버튼을 클릭 시
action속성에 작성한 URL로
method속성에 작성한 요청 방식을 통해
사용자가 input요소에 입력한 값들이 key=value세트로 전달

select태그와 option태그

- 사용자에게 키보드로 직접 입력받는 것이 아니라 제시한 여러 개의 옵션 중에서 한 가지를 선택할 수 있게끔 해주는 태그
- 제출 시 현재 선택된(selected)값이 넘어감
- 각각의 option요소에 value를 기술했을 경우 value속성값이 넘어가며, value속성이 존재하지 않는다면 option의 content영역의 값이 value로 넘어감
